### VSCode 단축키

- ctrl + 좌우

    공백 기준 커서 위치 옮기기

- ctrl + shift + 좌우

    공백 기준 드래그

- alt + shift 

    현재 커서 위치 라인 복제

- ctrl + d

    현재 드래그(포커싱)된 문자열과 일치하는 문자열 같이 드래그 

- ctrl + alt + 위아래 방향키 

    커서 증식

- ctrl + 위아래 방향키 

    스크롤

- alt + 위 아래 방향키
  
    포커싱 된 라인 위치 옮기기

- ctrl + tab

    현재 활성화된 tab 위치 바꾸기 

- ctrl + b

    explorer 

    markdown 적을 때는 **강조** 된다? 

- alt + 방향키 

- ctrl + w

    tab 종료
