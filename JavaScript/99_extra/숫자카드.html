카드 개수 문제

0에서 9까지 숫자가 적힌 N장의 카드가 주어집니다.

가장 많은 카드에 적힌 숫자와 카드가 몇 장인지 출력하는 프로그램을 만드세요. 카드 장수가 같을 때는 적힌 숫자가 큰 쪽을 출력합니다.

입력 형식

  • 첫 줄에 테스트 케이스 개수 T가 주어집니다. (1 ≤ T ≤ 50)
  • 다음 줄부터 각 테스트 케이스의 첫 줄에 카드 장수 N이 주어집니다. (5 ≤ N ≤ 100)
  • 다음 줄에 N개의 숫자 ai가 여백 없이 주어집니다. (0으로 시작할 수도 있음, 0 ≤ ai ≤ 9)

출력 형식

각 줄마다 "#T" (T는 테스트 케이스 번호)를 출력한 뒤, 가장 많은 카드의 숫자와 장 수를 차례로 출력합니다.

입력 예제

3
5
49679
5
08271
10
7797946543

출력 예제

#1 9 2
#2 8 1
#3 7 3
